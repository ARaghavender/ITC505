HTML Tags Description

  1. DOCTYPE html: This declaration specifies the document type and version of HTML being used, which is HTML5 in this case.
  2. html lang="en": Defines the root element of the HTML document and specifies the language of the content, which is English ("en").
  3. head: Contains meta-information about the HTML document, such as title, links to stylesheets, and scripts.
  4. link: Links an external CSS stylesheet (style.css) to the HTML document to define its presentation and layout.
  5. style: Defines internal CSS styles within the HTML document. In this case, it's used to apply styles directly to elements within the document.
  6. body: Contains the content of the HTML document that's visible to users, including text, images, links, and other elements.
  7. section: Defines a section of the document with the class "about" and the ID "about". Classes and IDs are used to apply styles or target specific elements with CSS or JavaScript.
  8. div: A generic container element used to group and style content within the document.
  9. h2: Defines a level 2 heading with the class "title". Headings are used to structure and organize content hierarchically.
  10. img: Inserts an image (profile-1.jpg) with specified dimensions and an empty alt attribute for accessibility.
  11. p: Defines a paragraph of text.
  12. h4: Defines a level 4 heading.
  13. ul: Defines an unordered list, typically used to list items without any specific order.
  14. li: Defines a list item within an ordered or unordered list.
  15. a: Creates a hyperlink that directs users to another page (addendum.html) when clicked.
  16. footer: Defines a footer for the document or section. Typically contains information such as copyright, authorship, or contact details.
