Hover on a Font

Serving Font

I have researched fonts from various sites and selected several from Google Fonts, including 'Pacifico', 'Lobster', 'Great Vibes', 'Indie Flower', 'Quicksand', 'Amatic SC', 'Righteous', 'Cookie', 'Anton', and 'Dancing Script'. These fonts can be imported via a URL from the Google Fonts website. In the CSS, I used a class called .font-sample to style the font samples. This class positions the text absolutely at the center of its container, sets the font size to 40px, and initially hides the text by setting its opacity to 0. The opacity transitions to 1 when the element is hovered over, creating a fade-in effect. Labels are styled as block elements with a margin to ensure proper spacing. Using a CSS sibling selector, the font sample becomes visible when the label is hovered over. For specific fonts, additional classes (like .Pacifico) are created to apply the respective font-family and text color. In the HTML, labels and font samples are structured so that hovering over a label triggers the display of the associated font sample, providing an interactive preview of the fonts. This setup allows users to see how each font looks in a smooth and visually appealing manner.
