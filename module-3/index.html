Designing of 3D VR Website

The website leverages 3D VR technology with speech recognition for enhanced interaction, allowing users to perform tasks like opening Wikipedia and Chrome using voice commands. It simplifies VR setup, requiring no installations, and improves client interaction by clearly demonstrating the site's pros and cons. Voice-controlled website

The project aims to automate website navigation using voice commands, making browsing effortless and reducing manual interaction. Developed primarily with JavaScript and CSS, it utilizes the annyang library for voice recognition and the Bootstrap framework for responsive design. This integration allows users to control the website through simple voice instructions, enhancing accessibility and user experience.
